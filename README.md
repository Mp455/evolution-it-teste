## 💻 Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Mp455/evolution-it-teste.git
```

Entre no diretório do projeto

```bash
  cd evolution-it-teste
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
